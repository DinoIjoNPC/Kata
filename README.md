Aplikasi web Word Finder adalah kamus digital berbasis web yang memungkinkan pengguna mencari kata berdasarkan awalan tertentu. Dibangun dengan HTML, CSS, dan JavaScript murni (tanpa framework) dengan database lebih dari ribuan kata bahasa Indonesia.

✨ Fitur Utama

· 🔍 Pencarian Kata Berdasarkan Awalan - Cari kata dengan mengetikkan awalan (contoh: "me", "a", "ber")
· 📊 Database Ribuan Kata - Koleksi lengkap kata bahasa Indonesia
· 🎨 Dual Theme - Mode Gelap (Hitam) dan Mode Terang (Putih)
· ⚡ Responsif - Tampilan optimal di desktop maupun mobile
· 🔄 ReSearch Button - Reset pencarian dengan mudah
· 🧹 Deduplikasi Otomatis - Menghapus kata duplikat secara otomatis (tidak ada kata ganda di list)
· 📈 Live Counter - Menampilkan jumlah kata yang ditemukan dan total kata di database

🚀 Cara Penggunaan

1. Buka file index.html di browser manapun (Chrome, Firefox, Edge, dll)
2. Ketik awalan kata yang ingin dicari di kolom pencarian (contoh: "aku", "ber", "me")
3. Tekan Enter atau klik tombol Search
4. Lihat hasil pencarian yang muncul beserta jumlah katanya
5. Ganti tema dengan mengklik tombol "Hitam" atau "Putih" di pojok kanan atas
6. Reset pencarian dengan tombol ReSearch

🛠️ Teknologi yang Digunakan

· HTML5 - Struktur halaman
· CSS3 - Styling dan animasi (tanpa framework)
· JavaScript (Vanilla) - Logika pencarian dan interaktivitas
· No Dependencies - 100% murni, tidak perlu install apapun
